Robot Vacuum / Vacuum-Mop Combo Fault Detection & Repair Guide (200 Entries)

1. Symptom: Robot does not respond when powered on, indicator light is off. Detection: Check if the power adapter is firmly plugged in, if the power cable is damaged, and if the battery is properly installed. Fix: Replace with a working power cable, re-insert the battery, ensure the charging dock is powered on. If still unresponsive, contact after-sales for motherboard inspection.

2. Symptom: Robot shuts off immediately after powering on. Detection: Check if battery is fully depleted, if the unit is overheating, or if the battery is swollen. Fix: Fully charge and retry, move to a ventilated area to cool down. If the battery is swollen, replace immediately with an original battery.

3. Symptom: Robot cannot connect to WiFi. Detection: Confirm the phone and robot are on the same 2.4G network, verify the WiFi password is correct, and check if the WiFi module area is obstructed. Fix: Restart the router and robot, re-enter the WiFi password, wipe dust from the WiFi module area.

4. Symptom: APP cannot control the robot. Detection: Check if the network connection is normal, if the APP is the latest version, and if the robot is in standby mode. Fix: Switch networks, update the APP, restart the robot and re-bind the device.

5. Symptom: Robot frequently bumps into furniture during cleaning. Detection: Check if the obstacle avoidance camera/sensors have dust or water stains, and if the bumper strip is stuck. Fix: Wipe obstacle avoidance components with a dry cloth, press the bumper strip to confirm normal rebound, clear surrounding obstacles.

6. Symptom: Robot does not move or moves slowly. Detection: Check if the drive wheels or caster wheel are tangled with hair/debris, if the wheel assembly is stuck, or if the floor is too slippery. Fix: Clean tangled material from wheels, apply a small amount of specialized lubricant, test on a dry floor surface.

7. Symptom: Suction power has noticeably decreased. Detection: Check if the dustbin, filter, or air duct is clogged, if the main brush is tangled with debris, or if the suction inlet is blocked. Fix: Clean each blocked component one by one, cut tangled hair from the main brush, check that the suction inlet is clear.

8. Symptom: Main brush does not rotate. Detection: Check if the main brush is jammed by foreign objects, if the motor wiring is loose, or if the main brush motor is faulty. Fix: Clear debris from the main brush, reinstall it. If it still doesn't rotate, contact after-sales for motor inspection.

9. Symptom: Side brush does not rotate. Detection: Check if the side brush is properly installed, if the shaft is tangled with hair, or if the side brush motor is abnormal. Fix: Reinstall the side brush, clean shaft debris. If the motor makes unusual noises, replace the side brush motor.

10. Symptom: Robot cannot find the charging dock. Detection: Check if there are obstructions around the charging dock, if the return-to-charge sensor has dust, or if the charging dock infrared signal is normal. Fix: Clear obstacles within 3 meters of the charging dock, wipe the return-to-charge sensor, reposition the charging dock.

11. Symptom: Indicator light flashes during charging, unable to charge normally. Detection: Check if charging contacts are oxidized or dusty, if the power adapter is faulty, or if the battery is damaged. Fix: Wipe charging contacts with a dry cloth, replace with a compatible power adapter. If the battery is damaged, replace it.

12. Symptom: Battery life has significantly decreased. Detection: Check the battery's age, whether there has been overcharging or over-discharging, and if the battery is swollen. Fix: Avoid overcharging and over-discharging, periodically perform full charge-discharge cycles for calibration. If capacity drops below 50%, replace with an original battery.

13. Symptom: Incomplete mapping or map errors. Detection: Check if the environment has reflective surfaces, excessive moving obstacles, or sensor abnormalities. Fix: Cover reflective surfaces, clear moving obstacles, delete the old map and rebuild.

14. Symptom: Robot frequently pauses during cleaning. Detection: Check if the dustbin is full, if battery is low, or if sensors are blocked. Fix: Clean the dustbin, fully charge, wipe dust from sensor surfaces.

15. Symptom: Water tank leaks during mopping. Detection: Check if the water tank cap is tightened, if the seal ring is aged or detached, or if the water outlet pipe is clogged. Fix: Tighten the water tank cap, replace the aged seal ring, clear the water outlet pipe.

16. Symptom: No water output during mopping. Detection: Check if the water tank has water, if the flow rate is set to minimum, or if the water outlet pipe is clogged. Fix: Add clean water, increase the flow rate, gently clear the water outlet pipe with a toothpick.
